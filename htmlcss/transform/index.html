CSS Transform Properties

The transform property applies a 2D or 3D transformation to an element. This property allows you to rotate, scale, move, skew, etc., elements.

Syntax: none|transform-function |initial |inherit;

Matrix

matrix(n,n,n,n,n,n) = Defines a 2D transformation using a matrix of six values

matrix3d(n(16)) = Defines a 3D transformation using 4x4 matrix of 16 values

perspective

perspective = Defines a transformation that sets the distance between the user and the plane

syntax

transform: perspective()

Rotate

rotate = Defines a 2D rotation

syntax

transform: rotate(angle)

transform: rotate3d(x,y,z,angle) defines a 3D rotaion

transform: rotateX(angle) 3D rotaion along X-axis

tranform: rotateY(angle) 3d rotaion along Y-axis

transform: rotateZ(angle) 3D rotation along Z-axis

Translate

method moves an element from its current position (according to the parameters given for the X-axis and the Y-axis

syntax

transform: translate(x,y)2D translation

transform: translate3d(x,y,z)3D translation

transform: translateX(x)using only the value for the X-axis

transform: translateY(y) using only the value for the Y-axis

transform: translateZ(z) using only the value for the Z-axis

Scale

method increases or decreases the size of an element

syntax

transform: scale(x,y)scale transformation

transform: scale3d(x,y,z)3D scale transformation

transform: scaleX(x) Defines a scale transformation by giving a value for the Y-axis

transform: scaleZ(y)Defines a 3D scale transformation by giving a value for the Z-axis

Skew

method skews an element along the X-axis by the given angle.

syntax

tranform: skew(x-angle,y-angle) Defines a 2D skew transformation along the X- and the Y-axis

tranform: Defines a 2D skew transformation along the X-axis

tranform: skewY(angle) Defines a 2D skew transformation along the Y-axis

Multiple Functions

syntax

tranform: translate() rotate() translateY() .........

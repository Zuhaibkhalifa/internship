CSS position property

CSS syntax

position: static | absolute | fixed | relative | sticky | initial | inherit ;

static

Default value elements render in order as they appear in the dpcument flow

absolute

The element is positioned relative to its first positioned ancestor element

fixed

The element is positioned relative to the browser window always

relative

The element is positioned relative to its normal position

sticky

The element is positioned based on the users scroll position

initial

Sets the property to its defaul value

inherit

Inherits this propert from its parent element
